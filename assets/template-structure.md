```javascript
// 頁面設定：A4 尺寸
```

## 文件結構

### 封面區塊

```
Dream Big元大公益圓夢計畫
第九屆單位申請書

申請單位全銜：_______________________________________________
申請計畫 (請填寫貴單位圓夢計畫名稱): _________________________
```

### 表格一：計畫內容表

**表格設定**：兩欄式表格，左欄為欄位名稱（背景色），右欄為填寫內容

| 欄位 | 欄位名稱 | 內容說明 |
|-----|---------|---------|
| 1 | 服務區域概況及需求說明 | 多段落文字區 |
| 2 | 服務目的（預期藉由計畫解決之問題） | 多段落文字區 |
| 3 | 請條列單位目前「所有」服務項目與內容說明 | 條列式文字區 |
| 4 | 請勾選本次計畫可對應到之聯合國永續發展目標 | 17 項勾選框 |
| 5 | 請詳述預計申請 Dream Big 經費所進行的服務項目 | 多段落文字區 |
| 6 | 呈上題，請說明過去所產生的「社會效益」或「正面影響力」 | 多段落文字區 |
| 7 | 期望元大金控集團暨子公司之企業志工、元大文教基金會或其他資源為您的計畫提供什麼協助或合作？ | 多段落文字區 |
| 8 | 預期計畫成效 | 至少三項目標 |
| 9 | 請說明計畫成果預計發表方式與完成時間 | 多段落文字區 |
| 10 | 單位現有之宣傳資源或管道 | 條列式文字區 |
| 11 | 服務現場照片或示意/設計圖 | 8 格圖片+說明 |

### 表格二：計畫預算表

**表格設定**：5 欄式表格

```
| 項目 | 單價 | 數量 | 金額 | 備註 |
|------|------|------|------|------|
| [項目1] | [金額] | [數量] | [小計] | [說明] |
| [項目2] | [金額] | [數量] | [小計] | [說明] |
| ... | ... | ... | ... | ... |
| 預算金額總計 ||| [總金額] ||
```

**註記**：
- 【註】以上項目可視需求自行加列
- 相關會議與活動均依衛生福利部疾病管制署規定或實際狀況彈性辦理

## 表格樣式建議

### 欄位標題儲存格
```javascript
shading: { fill: "D5E8F0", type: ShadingType.CLEAR },  // 淺藍色背景
borders: { /* 單線邊框 */ },
margins: { top: 80, bottom: 80, left: 120, right: 120 }
```

### 內容儲存格
```javascript
shading: { fill: "FFFFFF", type: ShadingType.CLEAR },  // 白色背景
borders: { /* 單線邊框 */ },
margins: { top: 80, bottom: 80, left: 120, right: 120 }
```

### 字型設定
- 標題：12pt 粗體
- 內文：11pt 標準
- 字型：標楷體或新細明體（中文）、Arial（英文數字）
